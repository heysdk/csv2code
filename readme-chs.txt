csv2code

========

csv2code是一个将csv文件转换为各种代码的工具。

csv2code使用nodejs编写。



我们提供了一个简单的web页面方便大家使用，如果您需要在本地搭建环境，可以下载全部代码以后，执行 

nmp install

然后启动服务

node bin/www

然后在浏览器中，输入 http://127.0.0.1:3000 使用。



我们提供一组快速转换脚本，可以方便的直接转换csv文件，具体可以参考samples/csv2php.js。

我们还提供了一个快速的项目整体转换脚本，配置一个projdef.csv即可直接转换一个完整的项目，具体可以参考samples/makephpproj.js。
